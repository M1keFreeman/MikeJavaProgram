Logical Operator

1)And - &&
2)Or - ||
3)NOT - !

* Logical operator works only with Boolean data types
* Logical operators will be used to combine multiple conditions

== > Table for And operator (&&)

      | value 1  &&  value 2 = Result |

      | true     &&     true = true   |

      | true     &&    false = false  |

      | false    &&     true = false  |

      | false    &&    false = false  |

      == > Table for OR operator (||)

            | value 1  ||  value 2 = Result |

            | true     ||     true = true   |

            | true     ||    false = true  |

            | false    ||     true = true  |

            | false    ||    false = false  |


      ==>Table for NOT operator (!)

            value = Result

            !true = false

            !false= true
